# 🚀 官方更新公告 (Changelog) - v0.6.0

## [v0.6.0] - 2026-04-09
### 核心调整与修复 (Core Fixes)
- 🔒 **私密密钥彻底脱敏**:
  - 全面清理了开发环境的 Shell 历史记录与 `.git/config` 文件中的 GitHub Personal Access Token (PAT)。
  - **安全性提升**: 现在打包流水线已完全依托 GitHub 仓库内部安全的 Secrets (`GH_PAT`) 进行认证授权，没有任何凭证会暴露在代码文件或页面中。

- 📝 **介绍页面与公告彻底隔离**:
  - **文档修正**: 修复了由于某些自动化脚本覆盖导致的“应用介绍页面”显示错误。
  - **全新应用介绍页**: `README.md` 已被重新排版为纯粹、精美的《官方产品功能介绍说明书》。
  - **发布说明解耦**: 现在，每次通过流水线生成的 **Windows (`.exe`)** 和 **Android (`.apk`)**，在发布到 Releases 版本号里面时，将会自动挂载本篇**专属更新描述文件**。主页的介绍将永远保持纯净！

- 🎯 **双端极简打包发布 (Win + Android)**:
  - 强化了 Electron 和 Capacitor 的构建配置。现在的流水线绝对且仅提供 `.exe` 和 `.apk` 的下载文件。
