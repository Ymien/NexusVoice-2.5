# 更新日志 (Changelog) - v0.5.25

## [v0.5.25] - 2026-04-09
### 新增特性 (Features)
- 📝 **文档分离与重排版**: 将《更新日志》从主应用介绍文件中完全剥离，统一存放到 `changelogs/` 目录中，防止更新公告污染核心的介绍文件。重新排版了 `README.md` 的应用描述。

### 修复与优化 (Fixes)
- 🔧 **精简构建流水线**: 在 `package.json` 的 Electron 构建选项（`build`）中，正式移除了不必要的苹果 (`mac` / `dmg`) 与 Linux (`AppImage`) 构建输出配置。现在系统将只专注于提供更纯净的 **Windows (`.exe`)** 和 **Android (`.apk`)** 应用构建环境，大大减少了流水线的体积与报错概率。
