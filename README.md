# NexusVoice - 智能多模态语音助手

NexusVoice 是一款集成了前沿大语言模型与实时语音交互的智能终端系统。它提供简洁、现代的图形界面，支持多模态输入（文本、语音）与实时流式输出，致力于为用户提供最自然、流畅的 AI 交互体验。

## 🌟 核心功能

- **智能语音对话**：内置高精度语音识别与自然人声语音合成（TTS），支持实时语音唤醒和打断，提供流畅的“类人”对话体验。
- **多模型支持**：支持配置和无缝切换多种主流大语言模型（如 DeepSeek, GLM-4, 豆包等），满足不同场景下的性能和逻辑需求。
- **动态 UI 引擎**：内置六种深度定制的精美主题（清爽浅色、沉浸深色、赛博霓虹、马卡龙色、新粗野主义、拟态玻璃），满足各种场景下的审美需求。
- **多模态视觉同步**：支持外置视频背景与 AI 语音播报状态联动，在播放语音时自动同步视频播放，提供更直观的视觉反馈。
- **数据隐私与安全**：API 密钥等敏感数据均由本地浏览器和私有后端处理，全面保障用户隐私。

## 🚀 快速开始

### 1. 环境准备
- 确保系统已安装 Node.js (v18+)

### 2. 安装与运行
```bash
# 安装依赖
npm install

# 启动本地开发服务器
npm run dev
```

### 3. 构建生产版本
```bash
npm run build
```

## 🎨 主题说明

系统内置了多种设计语言级别的 UI 主题：
1. **清爽浅色 (Light)**：极简白色卡片，大圆角与柔和阴影，适合日常办公。
2. **沉浸深色 (Dark)**：暗色背景配合高对比度文字，保护视力。
3. **赛博霓虹 (Neon)**：直角边框、霓虹发光阴影与赛博朋克配色。
4. **马卡龙色 (Macaron)**：超大圆角、粉嫩马卡龙色系与扁平化设计。
5. **新粗野主义 (Brutalist)**：强烈的黑白对比、粗边框、硬阴影，极具视觉冲击力。
6. **拟态玻璃 (Glass)**：全局毛玻璃磨砂效果，呈现高级半透明质感。
